# Каталог фильмов

Небольшое веб-приложение для просмотра каталога фильмов.
Реализовано в рамках тестового задания.

## Функциональность
- Просмотр списка фильмов
- Поиск по названию
- Просмотр карточки фильма
- Загрузка данных с сервера
- Обработка состояний загрузки и ошибок
- Адаптивная верстка

## Используемые технологии
- React
- Vite
- React Router
- SCSS
- json-server

## Запуск проекта
### 1. Установка зависимостей
``npm install``
### 2. Запуск сервера
``npm run server``
### 3. Запуск приложения
``npm run dev``
